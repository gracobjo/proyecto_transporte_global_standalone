# Datos de configuración SIMLOG

- **`rutas_red_simlog.yaml`** — Catálogo explícito de conexiones de la red híbrida (malla de hubs, enlaces hub–satélite y refuerzos entre subnodos). La UI de **Rutas híbridas** lo usa para listar tramos; las distancias se calculan con las coordenadas de `config_nodos.py`.
