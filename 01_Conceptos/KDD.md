# KDD - Knowledge Discovery in Databases

## 🧠 Definición
Proceso de extraer conocimiento útil y desconocido de grandes volúmenes de datos.

## ⚙️ Cómo funciona
Proceso iterativo con las siguientes fases:

1. **Selección** - Elegir datos relevantes
2. **Preprocesamiento** - Limpieza y transformación
3. **Transformación** - Reducción de dimensionalidad
4. **Minería** - Aplicar algoritmos
5. **Evaluación** - Interpretar resultados

## 📊 Uso en SIMLOG
- Pipeline de análisis de datos de transporte
- Detección de anomalías
- Análisis de grafos
- Predicción de retrasos
