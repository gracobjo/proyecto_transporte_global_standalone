# 📚 Obsidian Vault - Sistema de Gestión del Conocimiento

## Overview

Este proyecto incluye un **vault de Obsidian** para documentar conceptos técnicos, proyectos, modelos de IA y guías del sistema SIMLOG.

---

## Cómo Acceder

1. **Obsidian** → Open Vault → Selecciona carpeta del proyecto
2. **Principal**: `Bienvenido.md`
3. **Buscar**: `Ctrl+O`

---

## Estructura

| Carpeta | Contenido |
|---------|-----------|
| `01_Conceptos/` | Glosario técnico |
| `02_Proyectos/` | Proyectos |
| `05_Snippets_Code/` | Código reutilizable |
