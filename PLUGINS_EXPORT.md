# 🔌 Plugins de Exportación Recomendados

## Instalar desde Obsidian (Settings → Community Plugins → Browse)

### 1. **Obsidian Git** (ya instalado)
- Sincroniza vault con Git
- Config: `.obsidian/community-plugins.json`

### 2. **HTML Export** (recomendado)
Busca: `obsidian-html`
- Exporta vault a HTML estático
- Configuración simple
- Funciona offline

### 3. **Quartz** (para web pública)
Sitio: https://quartz.jzhao.xyz
- Generador de sitio estático
- Hosting gratuito en GitHub Pages
- Ver `QUARTZ_SETUP.md`

---

## Alternativa: Publish (oficial)

Obsidian → Settings → Publish
- Hosting oficial de Obsidian
- Requiere suscripción (~$10/mes)
- URL: `tu-vault.obsidian.md`
