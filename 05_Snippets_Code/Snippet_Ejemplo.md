# Snippet_Ejemplo

## 📝 Descripción
Snippet de código de ejemplo.

## 💻 Código
```python
def funcion_ejemplo(parametro):
    """
    Función de ejemplo para demostrar estructura de snippets.
    
    Args:
        parametro: Descripción del parámetro
    
    Returns:
        Descripción del retorno
    """
    resultado = parametro * 2
    return resultado
```

## 🔧 Requisitos
- Python 3.8+

## 📌 Uso
```python
resultado = funcion_ejemplo(5)
print(resultado)  # 10
```
